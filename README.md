# seller App
*vue 2.0 练习项目，点餐App*
## 技术栈
> vue2 + vuex + vue-router + webpack + ES6/7
## 项目运行
> git clone https://github.com/codeyoyo/seller.git  
> cd seller  
> npm install  
> npm run dev
